# 漫画読破履歴アプリ

## 今回の更新
- ホーム画面アイコンを「コミック本の上に『漫』」デザインに変更
- コミカライズの有無（原作種別）を追加
  - コミックオリジナル
  - ノベル原作コミカライズ
  - その他原作あり
  - 不明
- 「今日読んだ話数」「今週読んだ話数」が読書記録から自動集計されるように変更
- 本棚 / 感想一覧 / 読書履歴 の切り替えを追加
- 本棚をジャンルごとに表示
- 途中作品に「付箋」を追加
  - 今何話まで読んだか
  - 次に読む時のメモ
- 本棚から「＋1」「＋3」「記録」で簡単に読書履歴を追加
- 読書履歴はあとから編集可能

## GitHub Pagesで更新するファイル
以下をリポジトリ直下にアップロードしてください。

- index.html
- manifest.webmanifest
- apple-touch-icon.png
- icon-192.png
- icon-512.png

既存の `index.html` は上書きでOKです。

## iPhoneのアイコンを新しくする方法
すでにホーム画面に追加済みの場合、アイコンがすぐ変わらないことがあります。
その場合は、古い「漫画読破履歴」をホーム画面から削除して、SafariでURLを開き直し、もう一度「ホーム画面に追加」してください。

## データについて
データはブラウザのローカルストレージに保存されます。
GitHubにファイルを上書きしても、同じiPhoneのSafari内の記録は基本的に残ります。
念のため、更新前にアプリ内の「データを書き出す」でバックアップしておくと安心です。
